# 🌿 Лаборатория растений

Интерактивная имитационная лаборатория для моделирования роста и развития растений в теплицах с настраиваемыми условиями среды.

Учебный проект по дисциплине «Технологическая практика» — АлтГТУ, направление 09.03.04 Программная инженерия.


## 🌱 Виды растений

| Вид | Особенность |
|-----|-------------|
| 🌸 Гербера | Цветение при 20–25°C и свете >70% |
| 🌳 Яблоня | Требует стратификации семени (2 дня при <5°C) |
| 🌵 Кактус | Гниёт при влажности >70%, любит жару и сухость |
| 🍅 Томат | 5 стадий включая плодоношение |
| 🌺 Орхидея | Самая капризная: штраф за каждый нарушенный параметр |

## ⚙️ Параметры среды

Для каждой теплицы независимо настраиваются:
- 🌡️ Температура (от −15 до +50°C)
- 💡 Освещённость (0–100%)
- 💧 Влажность воздуха (0–100%)
- 🪴 Влажность почвы (0–100%)
- 🌿 Питательные вещества / удобрения (0–100%)

## 🛠️ Стек технологий

- **React 18** — функциональные компоненты, хуки (useState, useEffect, useRef, useCallback)
- **Vite** — сборка и среда разработки
- **JavaScript ES2022** — классы, наследование, полиморфизм
- **CSS-переменные** — светлая тема без внешних фреймворков
